💡 CSS Variables

CSS variables (custom properties) allow you to store reusable values.

  • Defined using --var-name
  • Accessed with var(--var-name)
  • Can be updated with JavaScript

"Variables help keep your code DRY and maintainable."

📍 CSS Positioning

Positioning lets you control where elements appear on the page.

  • static – default flow
  • relative, absolute, fixed, sticky
  • Use top, left, z-index for placement

"Mastering position unlocks precise layout control."

🔲 CSS Grid

CSS Grid is a 2D layout system for rows and columns.

  • display: grid
  • grid-template-columns, grid-template-rows
  • gap, grid-area, place-items

"Grid simplifies complex, responsive layouts."

⚡ Transitions & Effects

Transitions make property changes smooth and animated.

  • transition: all 0.3s ease
  • Works with hover, focus, active, etc.
  • Pair with transforms for interactive UIs

"Smooth transitions create a better user experience."

🧱 Z-Index & Stacking

The z-index controls stacking order of overlapping elements.

  • Higher z-index = on top
  • Only works on positioned elements
  • Helps with modals, dropdowns, and tooltips

"Think in layers — z-index helps you organize them."

📐 Media Queries

Media queries adapt styles to different screen sizes.

  • @media (max-width: 768px) for tablets
  • Use breakpoints for phones, desktops, etc.
  • Essential for responsive design

"Media queries make your site device-friendly."

🎭 Pseudo-classes & Elements

Pseudo-classes and elements target states or parts of elements.

  • :hover, :focus, :nth-child()
  • ::before, ::after, ::placeholder
  • Useful for effects, highlights, and structure

"Target elements in creative and powerful ways."
