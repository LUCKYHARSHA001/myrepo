🎞️ CSS Animations

CSS animations allow elements to move, change, and transform over time.

  • @keyframes defines animation steps
  • animation-name, animation-duration, animation-iteration-count
  • Combine with transforms and transitions

"Bring life to your interfaces with CSS motion."

🌀 Scroll Snap

Scroll snapping locks scrolling to specific positions.

  • scroll-snap-type, scroll-snap-align
  • Great for sliders, carousels, or sections
  • Works on overflow containers

"Control scrolling like never before."

🧠 Advanced Selectors

Advanced selectors give precise control over styling.

  • :not(), :is(), :has()
  • [attribute*=value], [data-attr]
  • Chained combinators: div > p + span

"Advanced selectors let you style with precision."

🎚️ Custom Scrollbars

You can style scrollbars to match your theme.

  • Use ::-webkit-scrollbar, ::-webkit-scrollbar-thumb
  • Define width, color, hover effects
  • Mostly supported in WebKit browsers

"Even scrollbars deserve a little style."

🧪 Feature Queries

Feature queries let you apply styles based on browser support.

  • Use @supports (property: value)
  • Fallback for older browsers
  • Combine with modern layout techniques

"Gracefully degrade your styles across browsers."

🌙 Dark Mode

Adapt your design for light and dark system themes.

  • Use @media (prefers-color-scheme: dark)
  • Define separate variables or styles for dark mode
  • Enhances accessibility and comfort

"Modern sites respond to user preferences."

🎨 Blend Modes & Filters

Use blend modes and filters for creative effects.

  • mix-blend-mode, background-blend-mode
  • filter: blur(), brightness(), grayscale()
  • Create dramatic or subtle visual enhancements

"Visual magic with blend modes and filters."
