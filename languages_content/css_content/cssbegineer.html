🎨 What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages.

  • Controls colors, spacing, fonts, and layout.
  • Works with HTML to create visually appealing sites.
  • Styles are applied via selectors and rules.

"CSS brings design to your web pages."

🔍 CSS Selectors

Selectors target HTML elements to apply styles.

  • element – targets by tag name
  • .class – targets by class name
  • #id – targets by unique ID
  • * – universal selector (all elements)

"Selectors are how CSS knows what to style."

🌈 Colors & Backgrounds

CSS allows you to style elements with colors and background images.

  • color – text color
  • background-color – background fill
  • background-image – apply images
  • Use HEX, RGB, HSL, or named colors

"A splash of color changes everything."

✍️ Text Styling

Style headings, paragraphs, and fonts.

  • font-size, font-family
  • text-align, text-transform
  • line-height, letter-spacing

"Typography sets the tone of your content."

📦 Box Model

The box model defines how elements are spaced and sized.

  • content, padding, border, margin
  • Use box-sizing: border-box for predictable layout

"Every element is a box — learn to control it."

🧱 Layout with Flexbox

Flexbox is a powerful layout tool for arranging items in a row or column.

  • display: flex
  • justify-content and align-items
  • flex-direction, flex-wrap

"Flexbox makes layout easier and more flexible."

📱 Responsive Basics

Responsive design ensures your site looks good on all devices.

  • Use % and viewport units (vw, vh) for scaling
  • @media queries for screen-based styles
  • Combine with flexbox/grid for adaptive layouts

"A good site works everywhere — from phone to desktop."
