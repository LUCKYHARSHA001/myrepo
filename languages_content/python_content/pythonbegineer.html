📘 Introduction to Python

Python is a versatile, high-level programming language known for its readability and simplicity.

  • Created by Guido van Rossum in 1991.
  • Popular for web development, data analysis, AI, automation, and more.
  • Supports multiple programming paradigms including procedural and object-oriented.
  • Has a massive ecosystem of libraries and frameworks.

"Python is the easiest language to learn and the most powerful to use." – Anonymous

🔣 Basic Syntax & Structure

Python emphasizes code readability and uses indentation instead of braces.

  • No need for semicolons ; at the end of statements.
  • Indentation is crucial (usually 4 spaces).
  • Comments start with #.
  • Programs can be written quickly with fewer lines of code.

"Readability counts." – The Zen of Python

📦 Variables & Data Types

In Python, you don’t need to declare a variable’s type — it’s inferred automatically.

  • int – for integers (e.g., x = 5)
  • float – for decimal numbers (e.g., y = 3.14)
  • str – for text strings (e.g., name = "Alice")
  • bool – for true/false values (e.g., is_valid = True)

"In Python, simplicity is power."

🔌 Input / Output

Python makes input and output operations simple using built-in functions.

  • input() – reads user input as a string.
  • print() – displays output to the console.
  • You can format strings using f-strings: f"Hello, {name}".

"Your code should speak clearly — let Python help."

➕ Operators

Operators perform operations on variables and values.

  • Arithmetic: +, -, *, /, %, **, //
  • Comparison: ==, !=, >, <, >=, <=
  • Logical: and, or, not

"Think logically, operate efficiently."

🔀 Control Flow (if, elif, else)

Control flow allows you to make decisions in your program.

  • if – Executes a block if condition is true.
  • elif – Else-if for multiple conditions.
  • else – Executes if none of the conditions are met.

"Control your logic, and the program will follow."

🔁 Loops (for, while)

Loops let you repeat actions without rewriting code.

  • for – Iterates over a sequence (like a list or range).
  • while – Repeats as long as a condition is true.

"Repetition leads to mastery — use loops wisely."
