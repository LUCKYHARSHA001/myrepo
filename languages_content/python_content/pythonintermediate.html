🐍 Object-Oriented Programming

Python supports object-oriented programming with classes, inheritance, and encapsulation.

  • Create reusable blueprints using class.
  • Use __init__ to define constructors.
  • Supports single and multiple inheritance.

"In Python, everything is an object—even classes."

📦 Modules & Packages

Modules help organize Python code across multiple files, while packages group modules together.

  • Use import to include external modules.
  • Create your own module using Python files.
  • Packages are directories with an __init__.py file.

"Good code is modular and easy to reuse."

🚨 Exception Handling

Python uses try-except blocks to handle errors and keep programs running smoothly.

  • Use try, except, finally, and else.
  • Raise custom exceptions with raise.
  • Common exceptions: ValueError, TypeError, FileNotFoundError.

"Handle the unexpected—write safer Python code."

🧠 List Comprehensions

List comprehensions are concise ways to create lists using a single line of code.

  • Syntax: [expression for item in iterable if condition]
  • Works for lists, sets, and dictionaries.
  • Improves code readability and efficiency.

"Comprehensions turn loops into elegant expressions."

🗂 File Handling

Python allows reading from and writing to files using built-in functions.

  • Use open() with modes like 'r', 'w', and 'a'.
  • Read files using read(), readline(), or readlines().
  • Always close files with close() or use with open() to auto-close.

"Let your programs talk to the outside world through files."
