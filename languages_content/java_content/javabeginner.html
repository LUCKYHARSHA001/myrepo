📘 Introduction to Java

Java is a high-level, class-based, object-oriented programming language widely used for web, mobile, and enterprise applications.

  • Developed by Sun Microsystems in 1995, now owned by Oracle.
  • Platform-independent using the Java Virtual Machine (JVM).
  • Widely used in Android development, backend systems, and more.
  • Known for its simplicity, portability, and security.

"Write once, run anywhere." – Java Philosophy

🔣 Basic Syntax & Structure

A Java program is made up of classes and methods. Every application must have a main() method.

  • Class names start with uppercase, methods with lowercase.
  • Statements end with a semicolon ;.
  • Braces { } define code blocks.
  • Comments: // for single-line, /* */ for multi-line.

"Start with small steps and build great things."

📦 Variables & Data Types

Variables in Java are strongly typed. You must declare the type before using them.

  • int – for integers
  • double – for decimal numbers
  • char – for single characters
  • boolean – for true/false values

"In Java, every variable has a type — just like everything has a purpose."

🔌 Input / Output

Java uses the Scanner class for input and System.out.println() for output.

  • Scanner input = new Scanner(System.in); to take user input.
  • System.out.println() prints output with a new line.
  • System.out.print() prints without a new line.

"Input feeds the code, output speaks the result."

➕ Operators

Java supports various operators for arithmetic, comparison, and logic operations.

  • Arithmetic: +, -, *, /, %
  • Relational: ==, !=, >, <, >=, <=
  • Logical: &&, ||, !

"Operators are Java’s way of helping you think like a machine."

🔀 Control Flow (if, else, switch)

Java uses control statements to direct the flow of execution.

  • if – Checks a condition and executes code if true.
  • else – Executes alternative code if condition is false.
  • switch – Executes one case from many options based on a value.

"Code control is life control."

🔁 Loops (for, while, do-while)

Java provides looping constructs to execute code repeatedly based on conditions.

  • for – Best when number of iterations is known.
  • while – Runs as long as the condition is true.
  • do-while – Executes code block at least once before checking condition.

"In loops, we learn. In repetition, we master."
