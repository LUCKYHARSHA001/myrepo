🌐 What is HTML?

HTML (HyperText Markup Language) is the standard language for creating webpages.

  • Defines the structure of web pages.
  • Uses elements called tags like <h1>, <p>.
  • Works with CSS and JavaScript for styling and interactivity.

"HTML is the skeleton of your webpage."

🏗️ HTML Page Structure

An HTML document has a defined structure that browsers understand.

  • <!DOCTYPE html> – Declares document type.
  • <html> – Root element.
  • <head> – Metadata like title, styles.
  • <body> – Visible content of the page.

"Structure comes before style."

🏷️ Common Tags

Tags are building blocks of HTML. They wrap content and define its role.

  • <h1> to <h6> – Headings
  • <p> – Paragraphs
  • <div> – Container elements
  • <span> – Inline container

"Tags tell browsers what each part of the page means."

🔧 Attributes

Attributes provide additional information about HTML elements.

  • href – for links
  • src – for image sources
  • alt – alternative text for images
  • class and id – for styling and scripting

"Attributes give HTML more meaning."

🖼️ Images & Links

Use <img> and <a> to add visuals and navigation.

  • <img src="path" alt="text">
  • <a href="url">Click Here</a>

"A good webpage connects and communicates visually."

📋 Lists

HTML supports ordered and unordered lists.

  • <ul> – Unordered list (bullets)
  • <ol> – Ordered list (numbers)
  • <li> – List item

"Lists organize content clearly."

📝 Forms

Forms collect user input and send data to a server.

  • <form> – Defines a
  • <form> – Defines a form.
  • <input> – Gets text, email, password, etc.
  • <label> – Labels input fields.
  • <button> or <input type="submit"> – Submits the form.

"Forms are the gateway to interaction."
