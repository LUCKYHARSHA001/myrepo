ðŸ“œ Arrays & Objects

JavaScript arrays and objects are foundational data structures that store collections of values.

  • Arrays: Ordered lists of values like numbers, strings, etc.
  • Objects: Key-value pairs to store structured data.
  • Access with indexes (arrays) or keys (objects).

"Everything in JavaScript is an object."

ðŸ§© DOM Manipulation

The Document Object Model (DOM) represents the structure of a webpage, and JavaScript can manipulate it.

  • Use document.querySelector() to select elements.
  • Change content with innerHTML, textContent.
  • Add or remove elements dynamically.

"DOM lets you build interactive web experiences."

ðŸ“¤ Events & Listeners

JavaScript responds to user interactions through event listeners.

  • addEventListener attaches events like clicks or keypresses.
  • Can trigger custom functions when events occur.
  • Useful for dynamic interaction.

"Events drive interactivity in web apps."

ðŸ›  JSON & Local Storage

JSON is a format for storing and exchanging data. Local storage allows saving data in the browser.

  • JSON.stringify() and JSON.parse() convert data.
  • localStorage.setItem() stores data.
  • localStorage.getItem() retrieves data.

"Persist data across sessions using localStorage."

ðŸ§® Array Methods (map, filter, etc)

JavaScript offers powerful array methods for transforming and processing data.

  • map(): Transform each element.
  • filter(): Select elements based on a condition.
  • reduce(): Accumulate values into one result.

"Mastering array methods = mastering JS data processing."
