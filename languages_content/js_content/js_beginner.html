📘 Introduction to JavaScript

JavaScript is a versatile scripting language primarily used for creating interactive effects within web browsers.

  • Invented by Brendan Eich in 1995.
  • Runs in the browser, enabling dynamic web content.
  • Used for both front-end and back-end development (Node.js).

"JavaScript is the duct tape of the internet."

🔣 Syntax & Basics

JavaScript syntax is the set of rules that define a correctly structured JavaScript program.

  • Statements end with semicolons ;.
  • Case-sensitive and interpreted line-by-line.
  • Supports comments using // and /* */.

"Clean syntax means readable code."

📦 Variables (let, const, var)

Variables are containers for storing data values. JavaScript uses let, const, and var to declare them.

  • let: Block-scoped and reassignable.
  • const: Block-scoped and constant.
  • var: Function-scoped and older.

"Choose your variable type wisely."

➕ Operators

Operators perform operations on variables and values.

  • Arithmetic: +, -, *, /, %
  • Comparison: ==, ===, !=, <, >
  • Logical: &&, ||, !

"Operators give power to expressions."

🔀 Control Flow (if, else, switch)

Control flow allows you to direct the execution path of your program.

  • if, else if, and else statements.
  • switch statements for multiple conditions.

"Decisions shape logic."

🔁 Loops (for, while)

Loops execute a block of code repeatedly as long as a condition is true.

  • for loops for fixed iterations.
  • while loops for condition-based execution.
  • do-while executes at least once.

"Loop through logic like a pro."

🧠 Functions (Regular & Arrow)

Functions group reusable code blocks and can be declared in multiple ways.

  • Regular functions using function keyword.
  • Arrow functions using =>.
  • Can accept parameters and return values.

"Functions make your code DRY (Don't Repeat Yourself)."
