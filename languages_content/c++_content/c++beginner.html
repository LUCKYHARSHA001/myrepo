📘 Introduction to C++

C++ is a powerful, high-performance programming language that builds upon C, with added features like classes and objects.

  • Developed by Bjarne Stroustrup as an extension to C.
  • Supports both procedural and object-oriented programming.
  • Used in game development, software systems, and more.
  • Provides strong memory management and performance.

"C++ is a language that lends itself well to the development of large-scale software systems." – Bjarne Stroustrup

🔣 Basic Syntax & Structure

C++ programs typically consist of functions, statements, and header files.

  • Programs start with the main() function.
  • Uses #include directives for libraries.
  • Braces { } define blocks, and ; ends statements.
  • Supports both single-line and multi-line comments.

"Write code as if the person who maintains it is a violent psychopath who knows where you live."

📦 Variables & Data Types

Variables in C++ are declared with data types like int, float, string, and more.

  • int, float, double, char
  • bool for true/false
  • string from the <string> library

"Data types are the roots of clarity in code."

🔌 Input / Output (cin, cout)

C++ uses cin and cout for input and output, along with the <iostream> library.

  • cout prints to the console using <<
  • cin reads input using >>
  • Includes using namespace std; for simpler code

"Communication between user and machine begins with I/O."

➕ Operators

C++ supports a wide range of operators for performing calculations and logic.

  • Arithmetic: +, -, *, /, %
  • Relational: ==, !=, >, <, >=, <=
  • Logical: &&, ||, !

"Operators are the building blocks of logical expression."

🔀 Control Flow (if, else, switch)

C++ provides control statements for conditional logic and decision making.

  • if and else for decisions
  • switch for multiple conditions

"Code without control is chaos."

🔁 Loops (for, while, do-while)

Loops in C++ help in repetitive execution of code blocks.

  • for – definite loops
  • while – condition-based loops
  • do-while – executes at least once

"Repeat until mastery."

🧠 Functions

Functions help organize code into reusable blocks, enhancing readability and modularity.

  • Declared with return type, name, and parameters
  • void functions return nothing
  • Can be overloaded with different parameters

"Divide and conquer, one function at a time."
