🗃️ Arrays & Strings

Arrays in C++ are collections of elements of the same type. Strings are objects of the string class or character arrays.

  • Accessed using indices (0-based).
  • Standard library provides the string class for easier string handling.
  • Can perform operations like concatenation, comparison, and substring extraction.

"In C++, strings come alive with power and elegance."

📌 Pointers

Pointers are variables that store memory addresses. In C++, they are crucial for dynamic memory, object management, and low-level system access.

  • Use * and & for dereferencing and referencing.
  • Can be used with new/delete for memory management.
  • Used in polymorphism via virtual functions.

"Pointers are the bridge between high-level logic and machine-level control."

📄 Structures & Unions

Structures allow grouping different types of data. C++ extends this by enabling methods inside structures. Unions share memory between members.

  • Structures support member functions.
  • Unions are efficient when only one member is used at a time.

"Design data that reflects your domain—use structs wisely."

🛢 Dynamic Memory

C++ allows memory management through operators new and delete.

  • new allocates memory dynamically.
  • delete releases the allocated memory.
  • Smart pointers in C++11 automate memory handling.

"Dynamic memory gives your code the power to adapt."

👨‍🏫 Object-Oriented Programming (OOP)

OOP is a paradigm where code is organized into objects—instances of classes encapsulating data and behavior.

  • Supports Encapsulation, Inheritance, and Polymorphism.
  • Improves code modularity and reusability.
  • Classes define structure, objects are real-world instances.

"OOP is the heart of modern C++ programming."

🔗 Inheritance

Inheritance allows a class to acquire properties and behaviors of another class.

  • Base class: the original class; Derived class: the extending class.
  • Supports single, multiple, and hierarchical inheritance.
  • Promotes code reuse.

"Inheritance builds bridges across your class hierarchies."

🌀 Polymorphism

Polymorphism means many forms. It enables one interface to control access to different types of objects.

  • Compile-time: Function overloading, operator overloading.
  • Runtime: Virtual functions and dynamic dispatch.

"Polymorphism makes your code flexible and scalable."

📁 File Handling

C++ provides file stream classes like ifstream, ofstream, and fstream for file operations.

  • Open files using open() or constructor.
  • Read and write using stream operators.
  • Always close files with close().

"With file handling, your programs can remember."
