📂 File Handling

File handling in C allows reading and writing data to files stored on the disk.

  • Use fopen(), fclose() for opening and closing files.
  • fprintf() and fscanf() for formatted I/O.
  • Binary operations with fread(), fwrite().
  • Important modes: "r", "w", "a", etc.

"Reading and writing files is like speaking to your program's memory."

⚙ Preprocessors & Macros

Preprocessors in C handle directives like macros before compilation begins.

  • #define creates constants or inline functions.
  • #include includes header files.
  • #ifdef, #ifndef used for conditional compilation.

"Macros are powerful tools that shape your code at compile time."

🖥 Command Line Arguments

Command-line arguments allow passing parameters to your program from the shell.

  • Use int main(int argc, char *argv[]).
  • argc stores argument count.
  • argv is an array of argument strings.

"Command-line arguments make your program versatile and dynamic."

⛓ Linked Lists

A linked list is a linear data structure where each element points to the next.

  • Types: singly, doubly, and circular linked lists.
  • Nodes contain data and a pointer to the next node.
  • Used in dynamic memory allocation, stacks, and queues.

"Linked lists are the foundation of dynamic memory structures."

⏫ Data Structures (Stacks, Queues, Trees)

Advanced data structures help in organizing and processing data efficiently.

  • Stack: LIFO structure using push() and pop().
  • Queue: FIFO structure with enqueue() and dequeue().
  • Trees: Hierarchical structure useful in searching and sorting.

"Data structures define the efficiency of your logic."
