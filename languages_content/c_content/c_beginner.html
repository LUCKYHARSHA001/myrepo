📘 Introduction to C

C is a general-purpose programming language that has been the backbone of software development for decades.

  • Developed by Dennis Ritchie in the 1970s.
  • Used to build operating systems, embedded systems, and more.
  • Foundation for modern languages like C++, Java, and Python.
  • Offers low-level memory manipulation with high performance.

"C teaches you how the machine works." – Linus Torvalds

🔣 Basic Syntax & Structure

The structure of a C program includes functions, statements, and semicolon termination.

  • Every C program starts with the main() function.
  • Statements end with a semicolon ;.
  • Braces { } define blocks of code.
  • Comments use // for single-line and /* */ for multi-line.

"Start where you are. Use what you have. Do what you can."

📦 Variables & Data Types

Variables store data values, and each variable has a specific data type that defines its size and type.

  • int – for integers
  • float – for floating-point numbers
  • char – for characters
  • double – for large decimal numbers

"The way to get started is to quit talking and begin doing." – Walt Disney

🔌 Input / Output

Input and output are handled using the scanf() and printf() functions in C.

  • scanf() reads input from the user.
  • printf() displays output to the screen.
  • Format specifiers like %d, %f, %c are used.

"Your input defines your output. Choose wisely."

➕ Operators

Operators are symbols that perform operations on variables and values.

  • Arithmetic: +, -, *, /, %
  • Relational: ==, !=, >, <, >=, <=
  • Logical: &&, ||, !

"Operators are how we compute logic into life." – Anonymous

🔀 Control Flow (if, else, switch)

Control structures decide the flow of a program.

  • if – Executes a block if condition is true.
  • else – Executes if condition is false.
  • switch – Selects one of many code blocks to be executed.

"Control your code, control your results."

🔁 Loops (for, while, do-while)

Loops help execute a block of code repeatedly.

  • for – Loop with known iterations.
  • while – Repeats as long as condition is true.
  • do-while – Executes block at least once, then repeats.

"The loop of learning never ends."
