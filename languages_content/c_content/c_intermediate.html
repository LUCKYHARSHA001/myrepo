🧠 Functions & Recursion

Functions are blocks of code designed to perform a specific task, while recursion is when a function calls itself.

  • Improves code reusability and readability.
  • Function declaration, definition, and calling are key steps.
  • Recursive functions solve problems via repeated self-calls.
  • Used in problems like factorial, Fibonacci, etc.

"Recursion is the art of solving problems by solving smaller versions of the same problem."

🗃️ Arrays & Strings

Arrays store multiple values of the same type. Strings are arrays of characters ending with a null character.

  • Arrays have fixed size and type consistency.
  • Access elements using index (starting from 0).
  • Strings are handled using character arrays and string functions.
  • Useful in data grouping and manipulation.

"Strings are not just text—they are data structures."

📌 Pointers

Pointers store the memory addresses of variables and enable dynamic memory management and efficient array handling.

  • Use * to declare and dereference pointers.
  • Use & to get the address of a variable.
  • Essential for dynamic memory, arrays, and function arguments.

"Understanding pointers is the key to mastering C."

📄 Structures & Unions

Structures group variables of different types. Unions use shared memory for all members, saving space.

  • Use struct to define a structure.
  • Use union when only one variable is used at a time.
  • Helpful in organizing data logically.

"Structure your code like you structure your thoughts."

🛢 Dynamic Memory (malloc, free)

Dynamic memory allocation allows you to allocate memory at runtime using malloc(), calloc(), and free it with free().

  • malloc() allocates memory blocks.
  • calloc() allocates and initializes memory.
  • free() releases memory back to the system.
  • Essential for creating flexible data structures like linked lists.

"Memory management is what gives C its true power."
